## 🔧 **Getting Started**

### **For Readers**
1. Visit [inkly.space](https://inkly.space)
2. Browse categories or use the search function
3. Follow your favorite authors
4. Enable notifications for new content

### **For Contributors**
1. Create an author account
2. Submit your expertise areas
3. Follow our content guidelines
4. Start publishing high-quality articles

### **For Developers**
```bash
# Clone the repository
git clone https://github.com/inkly/inkly.space.git

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env.local

# Run development server
npm run dev
```
